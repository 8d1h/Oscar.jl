Elements

Elements are created by moving polynomials into the ring:

julia> R, x = PolynomialRing(QQ, :x=>1:3)
(Multivariate Polynomial Ring in x1, x2, x3 over Rational Field, fmpq_mpoly[x1, x2, x3])

julia> Q, mQ = quo(R, ideal([x[1]^2-2, x[2]^2-3, x[3]^2-5]))
(Quotient of Multivariate Polynomial Ring in x1, x2, x3 over Rational Field by ideal generated by: x1^2 - 2, x2^2 - 3, x3^2 - 5, Map from
Multivariate Polynomial Ring in x1, x2, x3 over Rational Field to Quotient of Multivariate Polynomial Ring in x1, x2, x3 over Rational Field by ideal generated by: x1^2 - 2, x2^2 - 3, x3^2 - 5 defined by a julia-function with inverse
)

julia> Q(x[2]+x[3])
x2 + x3

julia> Q(x[2])

One can also construct the integer $0$ with the empty constructor:

julia> ZZ()
0

The following special constructors are also provided:

  • zero(ZZ)
  • one(ZZ)
julia> zero(ZZ)
0

julia> one(ZZ)
1

Properties

  • sign(n::Oscar.Integer) -> Oscar.Integer

Returns the sign of n, i.e. $n/|n|$ if $n \neq 0$, or $0$ otherwise.

julia> sign(ZZ(23))
1

julia> sign(ZZ(0))
0

julia> sign(ZZ(-1))
-1

Basic arithmetic

Oscar provides the basic arithmetic operations +, - and * and comparison operators ==, !=, including mixed operations between Julia and Oscar integers. It also provides division and powering as described below.
